Tansto is a high-end wellbeing platform that provides professionals with the tools to manage their mental and physical health. The project features a bold, high-contrast design system and a seamless cross-platform experience.

I led both design and development, creating a typography-driven visual system that balances editorial content with actionable health tracking — resulting in a 40% engagement increase over industry benchmarks.
