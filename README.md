## Quick start

1. Clone the repository:
   ```bash
   git clone https://github.com/mahmoud-elgammal/melgammal.com.git
   cd melgammal.com
   ```

2. Install dependencies:
   ```bash
   bun install
   ```

3. Start the development server:
   ```bash
    bun run dev
    ```
